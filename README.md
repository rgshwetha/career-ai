# 🚀 Career AI Assistant

An intelligent career guidance web application that helps users explore career paths, improve skills, and get job recommendations — powered by a full-stack architecture.

---

## 🌟 Features

* 🤖 AI Career Assistant (chat-based guidance)
* 🎯 Career Recommendations
* 📚 Skill Suggestions
* 💼 Job Guidance
* 📊 Dashboard UI (React + Charts)
* ⚡ Fast and responsive interface

---

## 🧠 Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* React Router
* React Query
* Recharts

### Backend

* Node.js
* Express.js
* Axios

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/career-ai.git
cd career-ai
```

---

### 2️⃣ Run Backend

```
cd backend
npm install
node server.js
```

Backend runs on:

```
http://localhost:5000
```

---

### 3️⃣ Run Frontend

Open a new terminal:

```
cd career-ai
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

## 🔗 API Endpoint

```
POST /api/ai/chat
```

### Request:

```
{
  "message": "Suggest career options"
}
```

### Response:

```
{
  "reply": "You can explore careers like Software Engineer..."
}
```
